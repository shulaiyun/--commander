# 00_GDD_MVP

## 项目代号

**裂变纪元**

## 一句话定义

一款以 1894 - 1945 东亚与西太平洋为首发舞台的近现代战争沙盒策略 RPG。玩家可以在世界地图上游历、接任务、加入势力或自立、招募军官与专家、经营补给与工业，并亲自参与陆战、海战、空袭，逐步改写时代格局。

## 目标受众

### 核心用户
- 喜欢《三国大时代》式自由推进的人
- 喜欢历史战争题材的人
- 喜欢“占城 / 扩张 / 招将 / 成长”爽点的人
- 喜欢轻策略而非极硬核军模的人

### 次级用户
- 喜欢地图涂色和战报分享的人
- 喜欢阵营、军团和排名竞争的人
- 喜欢单机主体验 + 适度社交的人

## 产品支柱

1. **自由推进**
   - 玩家不是只看剧情，而是在地图上主动行动。

2. **占领快感**
   - 拿下一座港口、铁路枢纽、工业城，必须有明确反馈和收益。

3. **角色成长**
   - 玩家不是上帝视角数字管理器，自己也是一个能成长的角色。

4. **时代变化**
   - 科技、工业、战争方式和势力格局会变化。

5. **轻社交**
   - 分享、排名、军团与协作增加留存，但不打断单机核心。

## 核心循环

1. 在世界地图移动
2. 进入城市 / 港口 / 机场 / 枢纽
3. 接受任务、交涉、招募、补给
4. 组织部队和人物
5. 发起战役或响应事件
6. 进入战斗（陆 / 海 / 空）
7. 获得占领收益、人物成长、关系变化
8. 推动下一阶段版图与事件

## 首发范围

### 时间范围
- 1894 - 1945

### 地理范围
- 东亚 + 西太平洋

### 势力数量
- 3~4 个主要势力
- 2~3 个中立或地方势力

### 可玩内容
- 世界地图垂直切片
- 城市和节点占领
- 人物成长
- 势力关系
- 陆战垂直切片
- 海战 / 空战原型
- 事件链
- 轻社交原型

## 世界地图节点类型

- 首都
- 工业城
- 港口
- 铁路枢纽
- 机场
- 资源点
- 前线要塞

## 人物职业

- 指挥官
- 参谋官
- 舰长
- 飞行员
- 工程师
- 外交官
- 情报员

## 战斗层级

### 陆战
- 小规模区域战斗
- 强调掩体、推进、士气、火力点

### 海战
- 编队、炮击、鱼雷、航路控制

### 空战
- 护航、侦察、轰炸、拦截
- 不追求纯飞行模拟

## 最小社交方案

### 首发必须有
- 玩家档案
- 战绩勋章
- 排行榜
- 战报分享码
- 军团占位入口

### 首发先不做
- 大型实时 PVP
- 即时语音
- 开放世界同图多人
- 实时海陆空协同作战

## MVP 胜利条件

当以下体验都打通时，MVP 才算成立：

- 玩家能在地图上移动
- 玩家能进入 3 类节点
- 玩家能招募人物
- 玩家能占领节点
- 玩家能触发事件
- 玩家能打一场陆战
- 玩家能看到版图变化
- 玩家能把战绩进入排行榜或生成战报
