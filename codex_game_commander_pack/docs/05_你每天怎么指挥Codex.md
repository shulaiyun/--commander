# 05_你每天怎么指挥 Codex

## 你的角色

你是：
- 制作人
- 最终拍板人
- 范围控制者

Codex 是：
- 执行工程师
- 原型搭建者
- 文档同步者
- 自测协助者

## 你每天的标准动作

### 每天开始
1. 打开仓库
2. 让 Codex 先总结当前状态
3. 明确今天只做 1 个目标
4. 指定验收标准

### 中途
- 如果 Codex 跑偏，立刻收口
- 不允许“顺手把很多东西一起做了”
- 要求它更新计划和文档

### 每天结束
让 Codex 输出：
- 今日修改文件
- 今日功能
- 如何验证
- 已知问题
- 明天最小下一步

## 你对 Codex 说话的正确方式

### 错误说法
- 帮我把整个游戏做出来
- 你自由发挥
- 全部都一起做

### 正确说法
- 只做世界地图垂直切片，不接入联网
- 只做城市与势力的数据加载，不做 UI 美化
- 先给计划，再实现，不要扩大范围
- 做完后告诉我如何验证

## 你的铁律

1. 一次只做一刀
2. 先计划再编码
3. 每次都要可验证
4. 文档必须同步
5. 跑偏立即回滚
